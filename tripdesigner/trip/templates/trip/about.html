About TripPlanner

It is always fun to go on a trip, but a pain to make plans for it. You'll need to choose your destination, set the date, book the flight and hotel, figure out the locations of most popular tourist attractions and trendy restaurants. It is common to spend many hours, and even some money, reading through various travel guides, consulting a travel agency or Google your destination and try putting the pieces together from numerous travel notes.

TripPlanner is an intelligent personal trip plan designer. We can customize a detailed travel itinerary that provides recommendations for tourists attractions and accommodations, as well as other userful travel tips. It is optimized by machine learning algorithms to find the best arrangment.

This website is the class project for DS-GA 3001 - Advanced Python, which originates from a previous class project for DS-GA 1007 - Programming for Data Science. The front end is written in HTML, CSS and JavaScript, and backend deployed in Python and Django framework.
